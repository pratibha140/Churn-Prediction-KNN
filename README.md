# Customer Churn Prediction (KNN)

## Project Overview
This project predicts whether a customer will churn based on user behavior
It uses **K-Nearest Neighbour(KNN)** Machine Learnig Algorithm

## Dataset Features
- Age
- Gender
- Plan Type (Prepaid/Postpaid)
- Tenure
- Monthly Usage
- Churn

## Machine Learning Workflow
1. Data Loading using Pandas
2. Data Preprocessing
3. Model training using KNN
4. Model Evaluation
5. Prediction 

## Model Performance
- Algorithm: K-Nearest Neighbors (KNN)  
- Accuracy: **0.93**  
- Evaluation: Train-Test Split (80-20)
